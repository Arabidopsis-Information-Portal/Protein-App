This app displays protein data (in table format) obtained from ThaleMine , an Arabidopsis-specific instance of InterMine . InterMine is an open source data warehouse built specifically for the integration and analysis of biological data. The specific data displayed in the app includes curated comments (UniProt), features (UniProt), protein domain regions (InterPro), and publications associated with a protein (UniProt & NCBI).

Instructions

To get started, enter a protein identifier in the search box and click on the Search button. At this point, data will be loaded in five tabs labeled Summary, Curated Comments, Features, Protein Domain Regions, and Publications. The number of records found in each category will be displayed on the tab. The Summary tab displays summary protein information such as protein ID, name, source, length, molecular weight, synonyms, keywords, and UniProt information. The Curated Comments tab lists curated comments from UniProt. The Features tab lists the features of the protein from UniProt. The Protein Domain Regions tab lists all of the protein domain regions from InterPro. Finally, the Publications tab lists all of the publications associated with the selected protein.

Features (Tables)

  • Sort the table by any column by clicking on the column header.
  • Change the columns that are shown by clicking on the Column visibility button and then selecting/de-selecting the columns.
  • Export all of the data in the Curated Comments, Features, Protein Domain Regions, or Publications table in CSV or Excel format by clicking on the CSV or Excel button.
  • Export options:
    • When exporting CSV or Excel data, the file is saved without prompting in the browser's current download location and will be named as follows:
      • For Curated Comments data:
        • Comments_for-<query>.(csv/xlsx)
      • For Features data:
        • Features_for-<query>.(csv/xlsx)
      • For Protein Domain Regions data:
        • Protein_Domain_Regions_for-<query>.(csv/xlsx)
      • For Publication data:
        • Publications_for-<query>.(csv/xlsx)
    • NOTE: when using Safari for the Mac
      • Excel export may not work (Adobe Flash is required)
      • The file will be opened in a new tab or window and File->Save As... must be used to save the file.
  • Filter the results further by entering text in the Search box in the upper-right of the table.
  • Clicking on a PubMed ID will display the publication at NCBI.
