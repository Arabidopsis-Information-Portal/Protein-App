This app displays a table of proteins in ThaleMine. The table can be expanded to show additional information about the protein such as the primary identifier or the molecular weight.

How to use this app

When the table loads, it shows a list of every protein in ThaleMine. The first column is the protein identifier in the same format as the mRNA primary identifier in ThaleMine. The second column contains a button that allows the user to show more information about the associated protein. To show less information, just press the button again. Warning: there is an issue with the data where two proteins are incorrectly associated with the same mRNA primary identifier (e.g. AT2G46830.1).

You can search through the proteins using the search bar above the table on the right hand side. You can also sort the table by clicking the table headers. Below the table on the right hand side are all the pages that can be accessed. The number of proteins shown on each page can also be changed with the dropdown menu on above the table on the left hand side.
